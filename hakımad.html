Temel Başlangıç

Python programlama dilini öğrenmeye hoş geldiniz! Bu bölümde Python'un temellerini ele alacağız. Python, kolay okunabilir söz dizimi ve geniş kütüphane desteği ile güçlü bir programlama dilidir.

  • Python, dinamik bir dildir, yani değişken türlerini önceden belirtmenize gerek yoktur.
  • Yüksek seviyeli bir dil olduğu için insan tarafından kolayca okunabilir ve yazılabilir.
  • Hem yazılım geliştirme hem de bilimsel araştırmalar için kullanılır.
    # Python'da bir "Merhaba Dünya" örneği
    print("Merhaba Dünya")

Değişkenler

Python'da değişkenler, değerleri saklamak için kullanılır. Değişkenler herhangi bir veri tipinde olabilir. Örnekler:

    # Sayısal değişkenler
    yas = 25
    pi = 3.14
    print(yas, pi)
    
    # Metin değişkenleri
    isim = "Python"
    print("İsim:", isim)
    
    # Boolean (Doğru/Yanlış)
    kontrol = True
    print("Kontrol durumu:", kontrol)

Çıktı:

    25 3.14
    İsim: Python
    Kontrol durumu: True
            
    # Liste değişkenleri
    meyveler = ["Elma", "Muz", "Kiraz"]
    print("Meyveler:", meyveler)
    
    # Sözlük (Dictionary) değişkenleri
    kisi = {"isim": "Ahmet", "yas": 30}
    print("Kişi:", kisi)
    
    # Tuple değişkenleri
    koordinatlar = (10, 20)
    print("Koordinatlar:", koordinatlar)

Çıktı:

    Meyveler: ['Elma', 'Muz', 'Kiraz']
    Kişi: {'isim': 'Ahmet', 'yas': 30}
    Koordinatlar: (10, 20)
            
    # Set değişkenleri
    renkler = {"Kırmızı", "Mavi", "Yeşil"}
    print("Renkler:", renkler)
    
    # Kompleks sayılar
    kompleks_sayi = 2 + 3j
    print("Kompleks sayı:", kompleks_sayi)
    
    # None (Boş) değişken
    bos_deger = None
    print("Boş değer:", bos_deger)

Çıktı:

    Renkler: {'Kırmızı', 'Mavi', 'Yeşil'}
    Kompleks sayı: (2+3j)
    Boş değer: None

Döngüler

Döngüler, tekrarlayan işlemleri gerçekleştirmek için kullanılır. Python'da en yaygın döngü yöntemleri:

  • for döngüsü: Belirli bir aralıkta veya koleksiyonda gezinmek için kullanılır.
  • while döngüsü: Belirli bir koşul doğru olduğu sürece çalışır.
  # for döngüsü örneği
  for i in range(1, 6):
      print(f"Sayı: {i}")
  
  # while döngüsü örneği
  sayi = 0
  while sayi < 5:
      print(f"Değer: {sayi}")
      sayi += 1

Çıktı:

  Sayı: 1
  Sayı: 2
  Sayı: 3
  Sayı: 4
  Sayı: 5
  Değer: 0
  Değer: 1
  Değer: 2
  Değer: 3
  Değer: 4
          
  # Bir liste üzerinde gezinme
  meyveler = ["Elma", "Muz", "Kiraz"]
  for meyve in meyveler:
      print(f"Meyve: {meyve}")
  
  # Bir sözlük üzerinde gezinme
  kisi = {"isim": "Ahmet", "yas": 30}
  for anahtar, deger in kisi.items():
      print(f"{anahtar}: {deger}")

Çıktı:

  Meyve: Elma
  Meyve: Muz
  Meyve: Kiraz
  isim: Ahmet
  yas: 30
          
  # Döngü içinde koşullu ifadeler
  for i in range(1, 11):
      if i % 2 == 0:
          print(f"{i} çift bir sayıdır.")
      else:
          print(f"{i} tek bir sayıdır.")
  
  # Sonsuz döngüden çıkış (break kullanımı)
  sayi = 0
  while True:
      print(f"Sayı: {sayi}")
      sayi += 1
      if sayi == 3:
          print("Döngüden çıkılıyor.")
          break

Çıktı:

  1 tek bir sayıdır.
  2 çift bir sayıdır.
  3 tek bir sayıdır.
  ...
  Sayı: 0
  Sayı: 1
  Sayı: 2
  Döngüden çıkılıyor.

Koşullar

Koşullar, belirli bir duruma göre karar vermek için kullanılır. Örnek bir if-else yapısı:

# Koşul yapısı örneği
sayi = 10
if sayi > 5:
    print("Sayı 5'ten büyüktür")
else:
    print("Sayı 5'ten küçüktür")

Çıktı:

Sayı 5'ten büyüktür
        
# Birden fazla koşul kullanımı
not_degeri = 85
if not_degeri >= 90:
    print("Harf notu: AA")
elif not_degeri >= 80:
    print("Harf notu: BA")
elif not_degeri >= 70:
    print("Harf notu: BB")
else:
    print("Harf notu: CC")

Çıktı:

Harf notu: BA
        
# İç içe koşullar
kullanici_adi = "admin"
sifre = "1234"
if kullanici_adi == "admin":
    if sifre == "1234":
        print("Giriş başarılı!")
    else:
        print("Hatalı şifre!")
else:
    print("Kullanıcı adı bulunamadı!")

Çıktı:

Giriş başarılı!

Fonksiyonlar

Fonksiyonlar, kod tekrarını azaltmak ve daha okunabilir bir yapı oluşturmak için kullanılır. Örnekler:

# Fonksiyon tanımlama
def selamla(isim):
  print(f"Merhaba, {isim}!")

# Fonksiyon çağrısı
selamla("Ali")

Çıktı:

Merhaba, Ali!
      
# Toplama işlemi yapan fonksiyon
def topla(a, b):
  return a + b

# Fonksiyon çağrısı
sonuc = topla(5, 3)
print(f"Toplam: {sonuc}")

Çıktı:

Toplam: 8
      
# Varsayılan değerli parametre
def bilgi_ver(isim, yas=18):
  print(f"İsim: {isim}, Yaş: {yas}")

# Fonksiyon çağrıları
bilgi_ver("Ahmet", 25)
bilgi_ver("Zeynep")

Çıktı:

İsim: Ahmet, Yaş: 25
İsim: Zeynep, Yaş: 18

Modüller

Python modülleri, belirli işlevleri yerine getirmek için kullanılır. Örnekler:

# math modülünü kullanma
import math
print(math.sqrt(16))  # Çıktı: 4.0

Çıktı:

4.0
      
# random modülünü kullanma
import random
sayi = random.randint(1, 10)  # 1 ile 10 arasında rastgele bir sayı
print(f"Rastgele sayı: {sayi}")

Çıktı:

Rastgele sayı: (ör. 7)
      
# datetime modülünü kullanma
import datetime
bugun = datetime.date.today()
print(f"Bugünün tarihi: {bugun}")

Çıktı:

Bugünün tarihi: (ör. 2025-01-03)

Dosya İşlemleri

Python'da dosya işlemleri kolayca yapılabilir. Örnekler:

# Dosyaya yazma
dosya = open("ornek.txt", "w")
dosya.write("Merhaba, Python!")
dosya.close()

# Dosyadan okuma
dosya = open("ornek.txt", "r")
icerik = dosya.read()
print(icerik)
dosya.close()

Çıktı:

Merhaba, Python!
      
# Dosyaya birden fazla satır yazma
with open("veriler.txt", "w") as dosya:
  dosya.writelines(["Satır 1\n", "Satır 2\n", "Satır 3\n"])

# Dosyadan satır satır okuma
with open("veriler.txt", "r") as dosya:
  for satir in dosya:
      print(satir.strip())

Çıktı:

Satır 1
Satır 2
Satır 3
      
# Dosyaya ekleme ve kontrol
with open("gunluk.txt", "a") as dosya:
  dosya.write("Bugün çok güzel bir gün!\n")

# Dosyanın var olup olmadığını kontrol etme
import os
if os.path.exists("gunluk.txt"):
  with open("gunluk.txt", "r") as dosya:
      print(dosya.read())
else:
  print("Dosya bulunamadı!")

Çıktı:

Bugün çok güzel bir gün!
(önceki içerik varsa, onlar da görünecektir)

Hata Yönetimi

Python'da hataları yakalamak için try-except blokları kullanılır. Örnekler:

# Hata yakalama
try:
  sayi = int(input("Bir sayı girin: "))
  print(f"Girdiğiniz sayı: {sayi}")
except ValueError:
  print("Lütfen geçerli bir sayı girin!")

Çıktı:

Bir sayı girin: abc
Lütfen geçerli bir sayı girin!
      
# Dosya okuma hatası yakalama
try:
  with open("yok_olan_dosya.txt", "r") as dosya:
      icerik = dosya.read()
except FileNotFoundError:
  print("Dosya bulunamadı!")

Çıktı:

Dosya bulunamadı!
      
# İnteraktif kullanıcı hatası yakalama
try:
  sayi1 = int(input("Birinci sayıyı girin: "))
  sayi2 = int(input("İkinci sayıyı girin: "))
  sonuc = sayi1 / sayi2
  print(f"Sonuç: {sonuc}")
except ValueError:
  print("Lütfen geçerli bir sayı girin!")
except ZeroDivisionError:
  print("Bir sayıyı sıfıra bölemezsiniz!")
except Exception as e:
  print(f"Bilinmeyen hata: {str(e)}")

Çıktı:

Birinci sayıyı girin: 10
İkinci sayıyı girin: 0
Bir sayıyı sıfıra bölemezsiniz!

Sözlükler

Sözlükler, anahtar-değer çiftlerini saklamak için kullanılır. Örnekler:

# Sözlük oluşturma
sozluk = {"ad": "Ali", "yas": 25, "sehir": "Ankara"}
print("Ad:", sozluk["ad"])
print("Şehir:", sozluk["sehir"])
sozluk["meslek"] = "Mühendis"
print("Güncellenmiş Sözlük:", sozluk)

Çıktı:

Ad: Ali
Şehir: Ankara
Güncellenmiş Sözlük: {'ad': 'Ali', 'yas': 25, 'sehir': 'Ankara', 'meslek': 'Mühendis'}
      
# Sözlükten eleman silme
sozluk = {"ad": "Ali", "yas": 25, "sehir": "Ankara"}
del sozluk["yas"]
print("Güncellenmiş Sözlük:", sozluk)

Çıktı:

Güncellenmiş Sözlük: {'ad': 'Ali', 'sehir': 'Ankara'}
      
# Sözlükteki anahtarları ve değerleri alma
sozluk = {"ad": "Ali", "yas": 25, "sehir": "Ankara"}
for anahtar, deger in sozluk.items():
  print(f"Anahtar: {anahtar}, Değer: {deger}")

Çıktı:

Anahtar: ad, Değer: Ali
Anahtar: yas, Değer: 25
Anahtar: sehir, Değer: Ankara

Matematik İşlemcileri

Python'da matematik işlemleri kolayca yapılabilir. İşte bazı temel işlem örnekleri:

# Matematiksel işlemler
a = 10
b = 5

print(a + b)  # Toplama
print(a - b)  # Çıkarma
print(a * b)  # Çarpma
print(a / b)  # Bölme
print(a ** b) # Üüs alma
print(a % b)  # Mod alma

Çıktı:

15
5
50
2.0
100000
0
      
# Matematiksel fonksiyonlar
import math

x = 16
y = 2

print(math.sqrt(x))  # Karekök alma
print(math.pow(x, y))  # Üs alma
print(math.factorial(5))  # Faktöriyel hesaplama

Çıktı:

4.0
256.0
120
      
# Trigonometri fonksiyonları
import math

a = math.radians(45)  # Dereceyi radiana çevirme
sin_deger = math.sin(a)
cos_deger = math.cos(a)

print(f"Sin(45°): {sin_deger}")
print(f"Cos(45°): {cos_deger}")

Çıktı:

Sin(45°): 0.7071067811865475
Cos(45°): 0.7071067811865476

Dizi Değişkenleri

Dizi değişkenleri, birden fazla değeri tek bir değişkende saklamak için kullanılır. Örnekler:

# Liste (dizi) oluşturma
meyveler = ["Elma", "Armut", "Muz"]
print(meyveler[0])  # İlk eleman
print(meyveler[-1])  # Son eleman

# Eleman ekleme
meyveler.append("Karpuz")
print(meyveler)

# Eleman silme
meyveler.remove("Elma")
print(meyveler)

# Dizi uzunluğu
print("Dizideki eleman sayısı:", len(meyveler))

Çıktı:

Elma
Muz
['Elma', 'Armut', 'Muz', 'Karpuz']
['Armut', 'Muz', 'Karpuz']
Dizideki eleman sayısı: 3
      
# Dizi sıralama
meyveler = ["Elma", "Armut", "Muz", "Karpuz"]
meyveler.sort()  # Alfabetik sıralama
print("Sıralanmış dizi:", meyveler)

# Dizi tersine çevirme
meyveler.reverse()
print("Tersine çevrilmiş dizi:", meyveler)

Çıktı:

Sıralanmış dizi: ['Armut', 'Elma', 'Karpuz', 'Muz']
Tersine çevrilmiş dizi: ['Muz', 'Karpuz', 'Elma', 'Armut']
      
# Liste dilimleme
meyveler = ["Elma", "Armut", "Muz", "Karpuz", "Çilek"]
print("İlk üç meyve:", meyveler[:3])  # İlk üç eleman
print("Son iki meyve:", meyveler[-2:])  # Son iki eleman
print("Aradaki meyveler:", meyveler[1:4])  # 1. ile 3. indeks arasındaki elemanlar

Çıktı:

İlk üç meyve: ['Elma', 'Armut', 'Muz']
Son iki meyve: ['Karpuz', 'Çilek']
Aradaki meyveler: ['Armut', 'Muz', 'Karpuz']
